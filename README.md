# SIMS PPOB - Muhamad Andika Wardana

Proyek ini merupakan _technical assignment_ untuk posisi Front End Programmer. Tujuan dari proyek ini adalah untuk membangun sebuah aplikasi PPOB (Payment Point Online Bank) sederhana menggunakan React.js dan berinteraksi dengan REST API yang telah disediakan, sesuai dengan mockup yang diberikan.

## Live Demo

sims-ppob-muhamad-andika-wardana.vercel.app

## Fitur Aplikasi

Berikut adalah fitur-fitur yang diimplementasikan dalam aplikasi ini:

- **Registrasi Pengguna**
- **Login Pengguna**
- **Tampilan Profil Pengguna**
- **Update Data Profil**
- **Update Gambar Profil**
- **Top Up Saldo**
- **Proses Pembayaran Layanan**
- **Tampilan Riwayat Transaksi**

## Teknologi yang Digunakan

- **Vite** sebagai _build tool_
- **React.js** sebagai _library_ utama
- **React Router DOM** untuk _routing_ dan navigasi
- **Redux Toolkit** untuk _state management_
- **Tailwind CSS** untuk _styling_
- **Axios** untuk melakukan panggilan API

## Instalasi dan Cara Menjalankan Proyek

Untuk menjalankan proyek ini secara lokal, ikuti langkah-langkah berikut:

1.  **Clone repository ini:**

    ```bash
    git clone https://github.com/wrdnika/SIMS-PPOB-Muhamad_Andika_Wardana.git
    ```

2.  **Masuk ke direktori proyek:**

    ```bash
    cd SIMS-PPOB-Muhamad_Andika_Wardana
    ```

3.  **Install semua dependensi yang dibutuhkan:**

    ```bash
    npm install
    ```

4.  **Jalankan development server:**

    ```bash
    npm run dev
    ```

5.  Buka browser dan akses `http://localhost:5173` (atau port lain yang ditampilkan di terminal).
